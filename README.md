> 豆瓣日历(Calendouer) iOS 端，农历、事情提醒、实时天气、电影推荐。

## Features

- [x] 日历（阴历 | 阳历，节气，节日） 
- [x] 实时天气
- [ ] 备忘提醒
- [x] 电影推荐
- [ ] 图书推荐
